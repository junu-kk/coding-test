## input
```
triangle
[[7], [3, 8], [8, 1, 0], [2, 7, 4, 4], [4, 5, 2, 6, 5]]
```

## output
```
거쳐간 숫자의 최댓값
30
```

## notes

아래로 쭉 내려감.
```

## strategy
```
인덱스 i일때
하나 아래로 내려가면 i 또는 i+1만 접근 가능함.
그럼 dp_t를 n행까지의 합의 최대로 생각하면 되겠다...
라고 하면 또 좀 그렇네.

아래쪽 수의 입장을 먼저 들어보면
인덱스 i일때
윗기준i 혹은 i-1만 접근 가능하니

dp_t도 삼각형으로 생각하되
triangle[row][i] + max(dp_t[row-1][i-1], dp_t[row-1][i])
로 생각하면 되겠다!

맨 왼쪽, 오른쪽의 경우의 경계처리만 잘 해주자.
