## strategy
```
일단 큐쓰는건 거의 확실하고
시간으로 돌리기보단
다리 다 건널때까지 기다리는게 좋아보인다.

다 건널때까지:
    다리에 트럭이 있다면 직진 (체류시간 += 1)
    다 건넌 트럭이 있다면 다리무게에서 빼주고 (체류시간 == 다리길이일때 pop(0) 후 현다리무게 -= 트럭무게)
    트럭을 다리에 넣을 수 있다면 넣어주고 (현다리무게 + 새다리무게 <= 무게제한일때 삽입)
    시간 += 1
    
그러면 다리는 어떻게 표현할 것이냐.
젤 간단하게 : 리스트로 (무게, 체류시간)
어차피 전수검사 해야하니까 괜찮겠다.
```
